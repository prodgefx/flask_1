# 🧑‍💻 Developer Portfolio Website (Flask + HTML + CSS)

A fully responsive, professional-grade **developer portfolio website** built with **Flask**, **HTML**, and **CSS**. Designed for developers who want a clean, corporate-style online presence — complete with smooth scroll animations and a functional contact form that sends emails.

---

## 🚀 Features
- **Responsive Design:** Works seamlessly on desktop, tablet, and mobile.
- **Flask Backend:** Powers dynamic routing and contact form handling.
- **Animated Frontend:** Subtle fade-in scroll animations for an engaging user experience.
- **Email Contact Form:** Uses Flask-Mail to send messages securely.
- **Easy Deployment:** Ready for one-click deployment on Render or PythonAnywhere.

---

## 🏗️ Tech Stack
| Layer | Technologies |
|-------|---------------|
| Backend | Python, Flask, Flask-Mail |
| Frontend | HTML5, CSS3, JavaScript |
| Deployment | Render / PythonAnywhere / Localhost |
| Tools | Git, pip, Virtualenv |
